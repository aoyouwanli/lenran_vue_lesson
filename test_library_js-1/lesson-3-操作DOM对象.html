操作DOM对象演示

DOM就是文档对象模型：

整个浏览器就是一个DOM树形结构，包括各个div以及相应的url、a、href、button、input、select等，并且针对DOM，有以下几个操作：
更新：更新DOM节点
遍历DOM节点，得到DOM节点
删除：删除一个DOM节点
添加：添加一个DOM节点
要操作一个DOM节点，前提必须是先获得该节点，然后进行增删改；
获取一个DOM节点，要用document来进行操作，先按照以下代码新增一个div：
这是一个h6标签
this is label p1，用id命名
这是一个p2标签，用classname命名

我们通过document.getElementByTagName('h6');来获得h6标签的DOM节点；
我们通过document.getElementById('p1');来获得p1标签的DOM节点；
我们通过document.getElementsByClassName('p2');来获得p2标签的DOM节点；
我们通过dom_div = document.document.getElementById('show_div');来获得整个div的DOM节点，作为父节点；
要获取父节点下所有的子节点，可以通过children来实现，即child_dom_div = dom_div.children;来实现；
同理，既然能通过children来获取所有的子节点，那也可以获得第一个节点，最后一个节点，上一个节点等方式：
父节点有多个节点，以此演示：获取第一个节点方法：dom_div_first = dom_div.firstElementChild;
父节点有多个节点，以此演示：获取最后一个节点方法：dom_div_last = dom_div.lastElementChild;
通过p1节点，获取相邻的节点：获取p1的下一个节点方法：dom_p1_nextone = dom_p1.nextElementSibling;
