面向对象编程：

在以前，如果需要在function中，增加function
首先，假设我们有个user的function如下：

function single_user(name){ this.name = name; };

原有的继承，如果像在user的基础上新增一个function，假设为年龄age，需要新增一个prototype：

user.prototype.age = function (){ alert(this.user_name + ' age is xxx'); };

然后使用这个function：

var peter = new single_user('Peter');

那么用户姓名可以这样获得：

peter.user_name，用户年龄为：peter.user_age()"

ES6之后发生了变化

首先，需要新增一个类：以corp_user举例：class corp_user{ }；
需要通过constructor构造器内构造这个corp_user的属性，比如姓名:
class corp_user{

constructor(name){ this.corp_user_name = name; }


}
如果要在这个类中新增一个方法，比如说以age举例，则，可以直接在类当中，直接定义，连function这个关键字都不需要，如下：
class corp_user{

constructor(name){ this.corp_name = name; }

age(){ alert(this.corp_name + 'corp user age is yyy'); }

}
然后使用这个class：

var tom = new corp_user("Tom");

用户姓名可以这样获取：

tom.corp_user_name，用户年龄为：tom.age()
