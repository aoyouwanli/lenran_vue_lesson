面向对象编程：

在以前，如果需要在function中，增加function
首先，假设我们有个user的function如下：
function single_user(name){ this.name = name;alert(this.user_name); };
原有的继承，如果像在user的基础上新增一个function，假设为年龄age，需要新增一个prototype：
user.prototype.age = function (){ alert(name + ' age is xx'); };
然后赋值：peter = single_user('Peter');
演示：假设目前有个Peter的single_user，那么这个用户的信息如下：
用户姓名为：onclick="alert(user_name)"，用户年龄为：onclick="single_user('Peter').user_age()"

ES6之后发生了变化

首先，需要新增一个类：以corp_user举例：class corp_user{ }；
需要通过constructor构造器内构造这个corp_user的属性，比如姓名:
class corp_user{
constructor(corp_name){ corp_name = name; alert(corp_name); }
}
如果要在这个类中新增一个方法，比如说corp_user_age举例，则，可以直接在类当中，直接定义，连function这个关键字都不需要，如下：
class corp_user{
constructor(corp_name){ corp_name = name; alert(corp_name); }
corp_name_age(){ alert(name + 'corp user age is yyy'); }
}
假设有个Tom的corp_user，演示如下：
用户姓名为：onclick="alert(tom.corp_user_name)"，用户年龄为：onclick="tom.age()"
