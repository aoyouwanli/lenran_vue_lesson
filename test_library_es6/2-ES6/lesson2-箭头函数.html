箭头函数

箭头函数在未来的unipp以及脚手架中，会被大量使用；
比如以前的求和sum,是这样操作的：var sum = function(a,b){ return a+b;};
箭头函数使用后，第一种改进后，就是这样： var sum = (a,b)=>{ return a+b;};
第二种改进是： sum = (a,b) => a+b;

总体规律就是：1. 没有了function这个关键字；
2. 在入参后方增加箭头=>;
3. 如果逻辑代码，只有return，可以连return这个关键字和大括号也省略；
但是在箭头函数中，return前包含其他代码，比如var、if、let等，就不能省略return这个关键字；
4. 如果箭头函数中的入参，只有一个参数，你可以连入参的括号也省略，但是如果有多个参数，则不可以省略括号；
总结： example = fucntion(a){ return a*2}; （原始用法）
可以变为：example = (a)=>{ return a*2}; （箭头函数的改进）
可以变为：example = (a)=>a*2;
可以变为：example = a=>a*2;
