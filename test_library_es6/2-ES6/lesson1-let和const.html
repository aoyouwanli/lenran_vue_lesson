变量定义：

传统的变量定义都是使用var
比如var name = 'hello world';
比如var link = 'wwww.baidu.com';
再比如常量赋值： var pi = math.pi;
到了ES6，则是一般使用let进行赋值：
let name2 = name;let link2 = link;
常量赋值：const pi2 = math.pi;
通过F12，控制台可查看
注意：1. var和let的区别在于作用域，var的作用域是全局变量，let的作用域是局部变量；
以for循环作为案例演示： for(let或者var i=0; i < 5; i++){ console.log(i); };console.log(i);
比如在function中或者循环中，用let定义了变量，超出了这个function或者for循环，那么这个变量就失效报错了；
但是，如果是使用var定义的话，超出了这个function或者for循环，依然可以被使用；
2. 而且，let和const这两个赋值，是在ES6中被定义出来的，因此，let和const以及某些语法，在一些低版本的浏览器中不一定支持；
